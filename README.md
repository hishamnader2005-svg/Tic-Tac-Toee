# Tic Tac Toe (Java)

A simple Tic Tac Toe game built in Java. This project was created to practice core Java concepts such as control flow, arrays, object-oriented programming, and user interaction.

## Features

- Two-player game (Player X vs Player O)
- Turn-based gameplay
- Win detection (rows, columns, diagonals)
- Draw detection
- Clean and simple game logic
- Written entirely in Java

## Technologies Used

- Java (JDK 8 or higher recommended)
- Java Standard Library

## How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/tic-tac-toe-java.git
   
2. Navigate to the project directory
   cd tic-tac-toe-java

3. Compile the java files
   javac *.java

4. Run the game
   java control
